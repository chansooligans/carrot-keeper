# Assets

## cover.png

Title-screen and intro cover art. Drop in any image here as `cover.png` and
the game's TitleScene will use it automatically. If missing, the game falls
back to a procedural title.

Recommended dimensions: 640×576 or any 10:9 ratio (matches the canvas
display size). Higher-resolution images will be displayed at the canvas's
native size with smooth scaling via the HTML `<img>` overlay.
